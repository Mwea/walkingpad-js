Chrome (Recommended)

Web Bluetooth works out of the box. Just ensure you're on HTTPS or localhost.

Chrome on Linux

  1. Open chrome://flags
  2. Search for "Experimental Web Platform features"
  3. Set to Enabled
  4. Relaunch Chrome

You may also need to add your user to the bluetooth group:

sudo usermod -a -G bluetooth $USER

Firefox (Experimental)

  1. Open about:config
  2. Search for dom.webbluetooth.enabled
  3. Set to true
  4. Restart Firefox

Android

Enable Bluetooth and Location services. Grant permission when prompted.
